# Application-Testing---Comarch-Opaska-Zycia
Repozytorium zawierające ustandaryzowane przypadki testowe dla aplikacji "Comarch Opaska Życia".

## Informacje o projekcie
- **Aplikacja:** Comarch Opaska Życia
- **Dostępność:** Google Play (Android), App Store (iOS)
- **Autor:** Dawid Chryc

## Struktura Dokumentacji Testowej
Wszystkie scenariusze testowe zostały ujednolicone do szablonu zawierającego: ID, typ, priorytet, dane testowe, kroki, oczekiwany rezultat oraz miejsce na raportowanie defektów.

### Dostępne scenariusze:
1. `Scenariusz Testowy - Pobieranie`: Testy instalacji aplikacji.
2. `Scenariusz Testowy - Rejestracja`: Testy procesu zakładania konta.
3. `Scenariusz Testowy - Logowanie`: Testy autoryzacji użytkownika.
4. `Scenariusz Testowy - Użyteczność i Wydajność`: Testy środowiskowe (sieć, bateria, Bluetooth, UI).

## Uwagi dotyczące platform
Scenariusze są zoptymalizowane pod kątem testów manualnych na systemach Android oraz iOS. Należy uwzględnić drobne różnice w interfejsie systemowym (np. nawigacja wsteczna, komunikaty o uprawnieniach).
